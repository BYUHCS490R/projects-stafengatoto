Internet

Internet is a worldwide system that connects millions of computers. It also allows users to access, share and exchange informations like files and videos across the world.

Web

Web is a part of the internet that contains websites and pages you can visit using a browser, for example Safari. Web is also what message we are sending meaning html.

How are the Internet and Web different?

Internet is the big network that connects computers all over the world and the web is the part of the internet that has websites you can visit using a browser.

ASCII

A way for computers to represent letters, numbers, and symbols using numbers. It stands for American Standard Code of Information Interchange.

Unicode

Universal code that lets computers show text in any language.

What is UTF-8?

A code that lets computers display letters, symbols, and emojis from any language.

URL

Address of a website or page on the internet.

Domain Name

Name of a website that people type to visit.

HTTP

The set of rules computers use to send and receive web pages over the internet. HyperText Transfer Protocol.

HTTPS

Its like HTTP but it keeps your data safeby encrypting it while it travels between your browser and the website. HyperText Tranfer Protocol Secure

IP Address

Unique number asssigned to every device connected to the Internet like a home address for your computer or phone.

DNS

It translates website names or domain names into IP Address so computers can find them. Stands for Domain Name System.

W3C Recommendations

Guideline published by the World Wide Web Consortium for building websites and technologies.
