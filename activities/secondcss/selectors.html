CSS Selectors

Answers Engagement Questions

  1. Ways to select an item by tag name, by ID and by class.
  2. ID can only style one element and uses "#"", but Class can styles many elements and uses "."
  3. Span is an inline element and is used for small parts text and doesn't break to a new line. Div is used for bigger sections and always start on a new line.
  4. Div a general purpose container that has no specific meaning and simply organizes content. But Header/Main?Footer are meaningful semantic elements that indicate the function of the content.
  5. The diffrent ways to select a color are by name(red,blue), hex(ff00), rgb(233,0,0), rgba and hsl.
