The action attribute is a html form tag that tells the browser where to send the form's data when the user clicks submit.

The method attribute defines the way the data is sent either visible in the URL or hidden in the request lobby.

The GET tag you use it to retrieve or request information and the POST tag you use it for sending, change, add, update or to remove information.
